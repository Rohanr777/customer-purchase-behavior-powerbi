# 📊 Customer Purchase Behavior Analysis Dashboard (Power BI)

## Project Overview

This project presents an interactive Power BI dashboard analyzing customer purchasing behavior, sales trends, and payment insights. It explores how revenue varies across categories, countries, age groups, and time to support data-driven decision-making.

---

## Objectives

* Analyze overall sales performance and revenue trends  
* Identify top-performing countries and product categories  
* Understand customer behavior based on age groups  
* Evaluate payment method usage and transaction patterns  
* Examine monthly trends in transactions and revenue  

---

## Analysis Highlights

* Monthly revenue trend analysis  
* Category-wise revenue and transaction distribution  
* Country-wise performance comparison  
* Payment method usage and revenue contribution  
* Age group-based customer spending behavior  
* Average transaction value by payment method  

---

## Tools Used

* Power BI  
* DAX (Data Analysis Expressions)  
* Data Visualization Techniques  

---

## Dataset Information

The dataset includes:

* Transaction ID  
* User Name  
* Age  
* Country  
* Product Category  
* Purchase Amount  
* Payment Method  
* Transaction Date (Month, Year)
